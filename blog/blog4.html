(f)light

May 17, 2017

My favorite class this semester was 6.S08, Interconnected Embedded Systems. The class was a mix of electrical engineering and computer science. Over the course of the class we worked with a Teensy microcomputer to build mini electronic systems. For one of our labs, we used an accelerometer to create a basic step counter. For another one, we used a microphone and the Google Speech API to create a transcription service.

After a long brainstorming period, my final project team decided to create a simulated archery system with plastic bows and an LED target screen that we dubbed "(f)light". We 3D-printed a bow frame with very cool folding joints. We mounted a breadboard on one side of the bow with a Teensy and IMU sensor. String wire was run from one end of the bow to the other with a trigger button in the middle.

For the LED screen, we ordered a strip of 250 LED lights. We used a particleboard base and created a 15x15 matrix of square spaces with black foam to reflect the LED light inside. We soldered these lights together in a snake pattern from one corner of the board to the opposite one. Over the top we glued a piece of clear acrylic plastic.

When the user presses the trigger button, we capture the readings from the IMU accelerometer and gyroscope to calculate the angle at which the user is holding the bow. Based on those angles, we use projectile motion to model the theoretical trajectory towards the LED screen, assigning x and y coordinates to each matrix space for ease of calculation.

We designed the system as a game with five turns. Each turn had a different randomized target square on the matix that would light up in green. After a shot, the square corresponding to the user's bow angle would light up in red. The score for the shot was calculated as an inverse of the distance between these two squares.

Seeing this project through from a few bullet points on a whiteboard to a finished, functional product was really rewarding. Everything I had learned about wiring, a new programming language (C++), sensor signal reading, and more all came together for (f)light. Thanks to Professor Voldman for an awesome semester in 6.S08 and my teammates Caitlin, Lilia, and Sarah for a fantastic project experience.
