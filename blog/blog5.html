My Computer Setup

June 6, 2017

I just started working on a new computer this week, so I had the opportunity to review my configurations and redownload my favorite apps and tools. I listed some below, note this is geared towards Mac operating systems.

Homebrew

This is a great package manager that makes installations and updates easy.

Xcode command line tools are required for Homebrew, they can be installed at the command line with:

$ xcode-select --install

Then we can install Homebrew and add a path to its programs:

$ ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
$ echo 'export PATH="/usr/local/bin:$PATH"' >> ~/.bash_profile

Some straightforward brew commands:

$ brew install <package>
$ brew update
$ brew upgrade <package>
$ brew cleanup

Z Shell

I like zsh better than bash and I've made it my default shell. It can be installed like this:

$ brew install zsh zsh-completions
$ chsh -s $(which zsh)

The Oh-My-Zsh framework has awesome plugins and themes for zsh:

$ sh -c "$(curl -fsSL https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"

Customizations for zsh can be set in the .zshrc configuration file. I use the agnoster zsh theme.

Vim

The best editor for me in terminal. At first I liked it better than emacs just because constant use of the control key bothered me. But I've come to appreciate how simple, fast, and easy to customize it is.

The .vimrc file is where we can adjust default vim settings to our liking. I've used my file to turn on syntax highlighting, show line numbers, remap some of my key bindings, etc. It's available here.

I use pathogen to manage my vim plugins in a bundle folder:

$ mkdir -p ~/.vim/autoload ~/.vim/bundle && \
$ curl -LSso ~/.vim/autoload/pathogen.vim https://tpo.pe/pathogen.vim

Plugins can be installed like this:

$ git clone <link> ~/.vim/bundle/<name>

The ones I use are:

  • NERDTree: file tree navigation system
  • EasyMotion: quick navigation in a file
  • NERDCommenter: adding comments in lots of languages
  • Airline: nice, customizable status line
  • YouCompleteMe: code completion

Other Apps

Chrome

My favorite browser, especially for web development. (free)

iTerm

My preferred terminal application. I use the Monokai Soda colortheme (available here) and Meslo powerline font (available here). (free)

Atom

When I'm not using vim, Atom is my text editor of choice, especially for larger projects. The git integrations are awesome. (free)

Simplenote (Beta)

A streamlined note-taking app with markdown integration that syncs across devices. (free)

Todoist

To do list app with organization by project and due date. Keyboard shortcuts and natural language processing make the user experience very smooth. (free, premium $28.99/year)

Itsycal

Neat app that places calendar date and shortcut in the toolbar. (free)

Spectacle

Makes screen partition with window shrinking and placement quick with keyboard shortcuts. (free)

CheatSheet

Holding down the command key pops up a list of keyboard shortcuts. (free)
